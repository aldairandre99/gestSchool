# Registar

> Cria uma conta nova no sistema.

## Quem pode registar-se aqui

Esta página está aberta a quem cria a sua própria conta. Em escolas onde a Direcção controla o acesso, é normal o registo público estar desactivado — neste caso, contacta a Secretaria para que te criem a conta com o perfil adequado.

## Campos obrigatórios

- **Nome** completo
- **E-mail** — será o teu identificador único
- **Palavra-passe** + confirmação — mínimo 8 caracteres

## Depois de criar a conta

1. Vais receber um e-mail com link de **verificação**
2. Carrega no link para activar a conta
3. Faz login com o e-mail e palavra-passe que escolheste
4. A Direcção atribui-te o perfil (Professor, Encarregado, etc.)

## Notas

- Pessoal docente/administrativo geralmente recebe a conta criada pela Direcção, não usa este formulário.
- Os encarregados podem registar-se aqui se a escola permitir auto-inscrição.
