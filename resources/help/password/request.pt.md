# Recuperar palavra-passe

> Pede um link para definir uma nova palavra-passe.

## Como funciona

1. Escreve o **e-mail** da tua conta
2. Carrega em **Enviar link**
3. Verifica a tua caixa de correio (incluindo spam)
4. Clica no link recebido — abre uma página onde defines a nova palavra-passe

## Notas

- O link é válido por **60 minutos**. Se expirar, pede outro nesta mesma página.
- Se o e-mail não chegar em 5 minutos, verifica spam e confirma que escreveste o e-mail certo. Se mesmo assim não chegar, contacta a Secretaria.
- Por segurança, o sistema **não revela** se um e-mail existe ou não na base — vais ver sempre a mesma confirmação.
