# Painel da Direcção

> Vista geral da escola para Director Geral, Pedagógico e Secretário. Mostra os números-chave do ano lectivo activo e atalhos para os ecrãs mais usados.

## O que vês aqui

5 cartões com contadores principais, todos clicáveis:

| Cartão | O que conta |
|---|---|
| **Utilizadores** | Total de contas no sistema (todos os perfis) |
| **Pessoal Administrativo** | Funcionários da secretaria/direcção |
| **Corpo Docente** | Professores + Professores Assistentes |
| **Alunos** | Total de alunos com ficha aberta |
| **Encarregados de Educação** | Representantes legais registados |

## O que podes fazer

### Saltar para uma área
Cada cartão é um link directo para a listagem correspondente. Por exemplo, clicar em **Alunos** abre `/alunos` com toda a lista filtrável.

### Aceder a ecrãs avançados
Pelo menu lateral (sidebar) tens acesso completo a:
- **Estrutura Académica** — anos lectivos, classes, cursos, turmas, disciplinas, trimestres
- **Operação Pedagógica** — matrículas, atribuições, aulas, avaliações, pautas, horários
- **Calendário** — eventos escolares
- **Sistema** — gestão de utilizadores (só Director Geral)

## Dicas

- Os contadores incluem **todos os registos**, não filtrados por ano lectivo activo. Para ver dados do ano corrente, abre a listagem respectiva e usa os filtros.
- Para o Director Pedagógico, o ecrã com mais valor diário tende a ser **Horários** + **Pautas**.
- Para o Secretário, **Matrículas** + **Alunos** são o ponto de partida diário.

## Notas de regulamentação

A gestão da informação aqui mostrada está sujeita à **Lei n.º 22/11** (Protecção de Dados Pessoais) — todo o acesso é auditado e os dados sensíveis (BI, morada, observações) são guardados encriptados em repouso.
