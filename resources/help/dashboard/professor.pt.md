# Painel do Professor

> Vista personalizada para o professor: perfil profissional, turmas que lecciona, atalhos diários para aulas, presenças e notas.

## O que vês aqui

### Card "Perfil"

Os teus dados profissionais:
- **Nº de processo** — identificador único na escola
- **Habilitações** académicas
- **Disciplinas** que dás (vista geral em texto livre)

### Card "Atribuições"

Lista das tuas atribuições no ano lectivo activo (turma × disciplina). Cada linha tem um atalho para a **pauta** dessa atribuição no trimestre corrente.

### Outras secções (variam consoante a configuração)

- Próximas aulas
- Avaliações abertas / fechadas
- Comunicados destinados ao corpo docente

## O que podes fazer

### Lançar presenças
1. Vai a **Aulas** no menu lateral
2. Escolhe a aula do dia
3. Marca presenças/faltas dos alunos
4. Grava — fica registado com o teu nome

### Lançar notas
1. Vai a **Avaliações** ou usa o atalho da pauta nas Atribuições
2. Escolhe a avaliação aberta
3. Preenche notas dos alunos
4. A pauta actualiza-se automaticamente

### Ver o teu horário
Menu **Horários** → vê a tua agenda semanal. Tens acesso só aos horários das turmas onde leccionas ou és director de turma.

### Reportar uma falta tua
Só a Direcção pode registar formalmente uma falta ao serviço. Comunica primeiro a quem coordena pedagogicamente; podes consultar **Faltas dos Professores** para ver o teu histórico.

## Dicas

- O teu nome aparece sempre que registas uma presença ou nota — fica registado para auditoria.
- Se uma turma onde lecciona não aparece nas atribuições, pode haver problema de configuração: contacta o Director Pedagógico.
