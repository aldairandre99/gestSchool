# Painel

> Vista geral da escola para a direcção. Mostra números-chave do ano lectivo activo e atalhos para os ecrãs mais usados.

## O que vês aqui

- **Contadores principais**: total de alunos matriculados, professores activos, turmas abertas, funcionários
- **Próximos eventos** do calendário escolar
- **Comunicados recentes**
- **Resumo da actividade** das últimas aulas e avaliações

## O que podes fazer

### Saltar para uma área
Os cards do topo são clicáveis — abrem a listagem correspondente (Alunos, Professores, Turmas).

### Ver detalhe de um evento
Os eventos do calendário no painel são links. Clica para ir directo ao detalhe.

## Dicas

- Os números reflectem **apenas o ano lectivo activo**. Para histórico de anos anteriores, abre a página de Anos Lectivos.
- Se um número parece desactualizado, faz refresh da página (Ctrl+R) — os contadores são lidos da base de dados em tempo real, mas a sessão pode ter cache.
