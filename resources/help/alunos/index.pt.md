# Alunos

> Listagem de todos os alunos da instituição. Ponto de partida para criar fichas, consultar perfis, e ligar encarregados.

## O que vês aqui

- Tabela com **número de processo, nome, classe, turma e ano lectivo** de cada aluno
- Campo de **pesquisa** que filtra por nome ou número de processo
- Botão **Novo** para criar uma ficha (se tiveres permissão)

## O que podes fazer

### Pesquisar um aluno
Escreve no campo de busca. A pesquisa é por **nome** ou **nº de processo**. O autofill do browser está desactivado neste campo.

### Criar uma ficha nova
1. Clica em **Novo** (canto superior direito).
2. Preenche os dados pessoais e académicos.
3. Liga um ou mais encarregados na secção dedicada.
4. Carrega em **Guardar** — o aluno é criado e podes matriculá-lo de seguida.

### Ver o perfil completo
Clica em **Ver** na linha do aluno. Acedes à ficha com:
- Dados pessoais
- Encarregados associados
- Histórico de matrículas (todos os anos)
- Boletins emitidos

### Editar uma ficha existente
Clica em **Editar**. Campos sensíveis (BI, morada, observações) são guardados **encriptados em repouso** na base de dados — ninguém com acesso só ao SQL os vê.

## Dicas

- O **número de processo** deve ser único e segue a convenção da escola — não é gerado automaticamente.
- Para matricular o aluno num ano lectivo, vai a [Matrículas](/matriculas) depois de criar a ficha.

## Notas de regulamentação

- Os dados pessoais aqui geridos estão sujeitos à **Lei n.º 22/11** (Protecção de Dados Pessoais). O encarregado dá consentimento explícito no acto de matrícula.
- A idade mínima de matrícula na 1.ª Classe é **6 anos completos** (Decreto Presidencial 162/23, Art. 21.º).
